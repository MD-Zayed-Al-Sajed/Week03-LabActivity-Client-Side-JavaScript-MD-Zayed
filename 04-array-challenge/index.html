Array Challenge

Challenge 1

Enter numbers separated by commas to sort them in descending order:


User input a series of numbers separated by commas. Then the system processes the input by arranging the numbers in descending order, filling in any missing numbers within the range from the smallest to the largest number.

Challenge 2

Enter two arrays of numbers separated by commas:



The system combines two input arrays, removes duplicates, sorts the resulting array in ascending order, and displays the removed duplicates.
